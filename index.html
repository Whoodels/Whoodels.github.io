How to Join WhoodeSMP

Server Version: Minecraft 1.21.1

Server IP: play.whoodesmp.com

Step-by-Step Installation Guide

  1. Install Minecraft Fabric Loader:

    Go to the Fabric Loader download page and install the latest version for Minecraft 1.21.1.

  2. Install the Simple Voice Chat Mod:

    Download the Simple Voice Chat mod from CurseForge and place it in your Minecraft mods folder.

  3. Configure Your Mods:

    Ensure that both the Fabric API and Simple Voice Chat mod are in your `mods` folder.

  4. Join the Server:

    Launch Minecraft with the Fabric profile, click "Multiplayer," and add the server address: play.whoodesmp.com.
